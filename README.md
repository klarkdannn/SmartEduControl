
Getting Started

### **Prerequisites**
- Java 17+
- Spring Boot
- Maven
- PostgreSQL


 Configure application.yml:
- add your database info in the datasource property.
- add your email info in the mail property

Build the project:
```bash
mvn clean install
```

Run the application:
```bash
mvn spring-boot:run
```
